# Pinterest video downloader bot

A Telegram bot for downloading videos from [Pinterest](https://www.pinterest.com/) via a link.

## Using Guide

1. Clone the project:

```
git clone https://github.com/flatline-01/pinterest_video_downloader_bot.git
```

2. Obtain the bot token from [Bot Father](https://t.me/BotFather) and set it in the .env file.

3. Run maven install command:

```
mvn install -f *path to the pom.xml*
```

4. Execute jar file:

```
java -jar *path to the .jar file*
```
